# 🏀 NBA Player Performance Dashboard

An interactive dashboard that analyzes NBA player performance using **per-36 metrics**, **True Shooting %**, and **Usage Rate**.  
Built with **Python**, **Streamlit**, **Plotly**, and the official [NBA API](https://github.com/swar/nba_api).

## 📌 Features
- Fetches real-time NBA player stats for the current season
- Calculates advanced metrics:
  - Per-36 scoring, assists, rebounds
  - True Shooting %
  - Usage Rate
- Interactive scatter plot to explore efficiency vs. workload
- Player-specific bar chart for per-36 performance
- Filters and interactivity for better exploration
